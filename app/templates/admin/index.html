calzone Administrative Interface

This administrative interface is granted to any user with admin role. The first valid user to log on to the system is automatically set to the administrator. Others can be added as needed in the User table.

This interface will allow for chaning user roles, selecting zone names and hardware IO pins, change settings, etc.

Use the menus at the top to set up the system.

Note: Don't delete the only admin user or you can lock yourself out.
